# Introduction to C#

C# is a modern, object-oriented programming language developed by Microsoft. It's widely used for building desktop, web, 
and mobile applications with frameworks like .NET.

### Key Features of C#
1. **Object-Oriented**: Supports encapsulation, inheritance, and polymorphism.
2. **Type-Safe**: Minimizes runtime errors.
3. **Versatile**: Ideal for web, desktop, and mobile development.
4. **Rich Library**: Access to a wide range of APIs for faster development.

### Hello World Example

```csharp
using System;

class Program
{
    static void Main(string[] args)
    {
        Console.WriteLine("Hello, World!");
    }
}
